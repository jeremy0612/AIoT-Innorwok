# Fitness Mobile App using WISE-PAAS Platform

This is a fitness mobile app that is being built using the WISE-PAAS platform. The app will allow users to track their fitness goals, log their workouts, and connect with friends for added motivation and support.

## Features

- Track fitness goals: Users can set and track their fitness goals, such as running a 5K or losing 10 pounds.
- Log workouts: Users can log their workouts, including exercise type, duration, and intensity.
- Recommendation :  Suggestions are based on the user's characteristics, hobbies, interest and physical aspect.
## Getting Started

To get started with the app, follow these steps:

1. Install the app on your mobile device.
2. Create an account or sign in with your existing account.
3. Set up your fitness goals, physical status, interest, hobbies by filling in our survey.
4. Log your workouts and track your progress.
